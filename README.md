# Automatic-Railway-Crossing-Gate
⚙️ System Description

The system operates by detecting an approaching train using entry sensors placed before the crossing. Once the train is detected, the gate closes automatically, and warning indicators are activated.

While the train is passing through the crossing area, the gate remains closed to prevent vehicles from crossing. After the train exits and is detected by the exit sensor, the system opens the gate and returns to normal operation.

This logic ensures safe and automated railway crossing control without manual intervention.

🔌 Inputs & Outputs
> Inputs:
- Train Detection Sensor (Entry)
- Train Detection Sensor (Exit)
- Emergency Stop Button
> Outputs:
- Gate Motor (Open/Close)
- Warning Lamp / Signal Light
- Buzzer / Alarm
